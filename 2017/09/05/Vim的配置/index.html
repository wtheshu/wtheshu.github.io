设置自动缩进

本文内容转载自 阿铭Linux

centos系统，修改vim的配置文件 /etc/vimrc 添加如下内容：

  1. 打开 vimrc ，添加以下语句来使得语法高亮显示：
  • syntax on
  1. 如果此时语法还是没有高亮显示，那么在 /etc 目录下的profile文件中添加以下语句：
  • export TERM=xterm-color
  1. 解决方向键和退格键失效的问题（采用非兼容模式）
  • set nocompatible
  • set backspace=2
  1. 设置 Windows 风格的 C/C++ 自动缩进（添加以下 set 语句到 vimrc 中）
    1. 设置（软）制表符宽度为 4 ：
    • set tabstop=4
    • set softtabstop=4
    1. 设置缩进的空格数为 4：
    • set shiftwidth=4
    1. 设置自动缩进 ：即每行的缩进值与上一行相等；使用 noautoindent 取消设置：
    • set autoindent
    1. 设置 使用 C/C++ 语言的自动缩进方式：
    • set cindent
    1. 设置 C/C++ 语言的具体缩进方式 ：
    • set cinoptions={0,1s,t0,n-2,p2s,(03s,=.5s,>1s,=1s,:1s
    1. 如果想在左侧显示文本的行号，可以用以下语句：
    • set nu
